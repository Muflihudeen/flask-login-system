# Flask Login System with SQLite k

A Flask-based authentication system with SQLite database integration.

## Features
- User registration
- User login
- Dashboard page
- Logout functionality
- SQLite database storage
- Duplicate username validation

## Technologies Used
- Python
- Flask
- Flask-SQLAlchemy
- SQLite
- HTML
- CSS

## Project Structure

├── app.py

├── static/
│   └── style.css
│

## How to Run

1. Install dependencies

pip install flask flask_sqlalchemy

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
